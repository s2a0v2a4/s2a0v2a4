# Developer Stack & Skills

Ich bin ein leidenschaftlicher Softwareentwickler mit Erfahrung in Frontend, Backend, Cloud-Hosting und modernen Web-Technologien. Im Laufe der Jahre habe ich an Projekten gearbeitet, die von klassischen Web-Apps bis hin zu Cloud-basierten SaaS-Lösungen reichen.

## Programming Languages
Python, JavaScript, C, C++, Rust, Kotlin, TypeScript, HTML5, Adobe

## Frontend Frameworks & Libraries
React.js, Vue.js, Angular, Svelte, SvelteKit, Next.js, TailwindCSS  
Ich konzentriere mich auf moderne, komponentenbasierte Ansätze und sauberes UI/UX-Design.

## Backend Frameworks & Platforms
Node.js, Express.js, Spring  
Erfahrung mit RESTful APIs, Serverlogik und Microservices.

## Web Servers & Application Servers
Apache, Nginx  
Optimierung von Performance, Security und Deployment-Pipelines.

## Databases
MySQL, MongoDB  
Solide Kenntnisse in relationalen und NoSQL-Datenbanken, inklusive Optimierung von Queries und Datenmodellen.

## Cloud & Hosting
AWS, Azure, Netlify, Vercel  
Skalierbare Cloud-Architekturen, CI/CD-Integration und Serverless-Deployments.

## Tools & Testing
Figma, Jest, Mocha, Vitest  
Versionierung, UI-Prototyping und automatisiertes Testing gehören zu meinem Standardworkflow.
